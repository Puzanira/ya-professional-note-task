# Сервис заметок к заданию 2 олимпиады "Я профессионал"
 Доументация API: https://app.swaggerhub.com/apis-docs/Puzanira/ya-professional-note-task/1.0.0
 
# Стек технологий:
  * mongoDB
  * mongoose
  * express (node v12)
  
# Запуск приложения: 
  * `docker-compose build`
  * `docker-compose up`
  
# Запуск тестов:
  * `cd src`
  * `npm install`
  * `npm run test`
